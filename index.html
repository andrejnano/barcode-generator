Adjust settings & download

Defaults work well for most use cases. Only change these if you have specific needs.

Include IDs in filenames
Adds EAN and product ID from URL to filenames for easy searching.
Output format
SVG is vector (scales perfectly for print). PNG is raster (works everywhere).
QR code size
SVG is always 30 × 30 mm. Scale only affects PNG resolution.
QR damage resistance
If the QR code might get scratched or partially covered, use High.
Barcode size
SVG is always 37.29 × 13.13 mm (GS1 standard). Scale only affects PNG resolution.
Show numbers under barcode
The 13-digit number printed below the barcode lines. Usually required for retail.
